# Data Visualization Lab

This repository contains my Power BI lab assignment analyzing two raw datasets.

## Tools Used
- Microsoft Power BI Desktop

## Datasets
- Netflix Titles (CSV)
- HR Employee Data (Excel)

## Contents

📂 powerbi – Power BI project (.pbix)

📂 datasets – Raw datasets

📂 screenshots – Dashboard previews

📂 blog – Analysis blog

## Key Insights

- Netflix catalog is dominated by Movies.
- Content peaked around 2018.
- TV-MA is the most common rating.
- HR dataset shows ~24% employee attrition.
- Sales is the largest department.
- Visualizations also exposed raw data inconsistencies (e.g., the William Wyler anomaly).
